# pi-dashboard-demo
Hosting Projects from Raspberry Pi_11

This repository contains a small experimental project created for learning and personal exploration. The project involved hosting a simple webpage from a Raspberry Pi and experimenting with basic Python scripting and web deployment.

The files in this repository reflect hands-on practice and experimentation rather than production-ready code. Any included wordlists or scripts were used strictly for educational purposes in a controlled environment.

EXPLANATION
This program demonstrates how attackers perform an offline dictionary attack against hashed passwords. The password entered by the user is converted into a cryptographic hash using OpenSSL. John the Ripper then reads each candidate password from a wordlist, hashes each candidate using the same hashing algorithm and salt, and compares the resulting hash to the stored hash. If the generated hash matches the stored hash, John has identified the original password.
